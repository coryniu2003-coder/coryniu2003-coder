# Hi, I'm Cory

I work at the intersection of scientific software, computational materials and machine learning for scientific computing.

My recent work focuses on building reproducible workflows for simulation and data-intensive research: preparing reference data, running model workflows, deploying simulations in LAMMPS, and validating whether the resulting trajectories are physically credible.

## What I Build

- Scientific Python workflows for simulation data, trajectory analysis and figure generation.
- LAMMPS-based molecular dynamics setups for machine-learning simulation models.
- Reproducible benchmarking pipelines for computational materials models.
- Image-analysis pipelines for local structure and quality metrics.

## Featured Work

### Scientific ML Reliability Benchmarking

End-to-end benchmark for testing machine-learning models for molecular simulation as deployed molecular dynamics engines. The workflow compares model behaviour against VASP/AIMD references using MSD, RDF/PDF and trajectory-level diagnostics across multiple materials systems.

Models used: MACE, CHGNet, MatGL, MatterSim, ORB and SevenNet.

### LAMMPS Scientific ML Integration Portfolio

A collection of reproducible LAMMPS/ML-GNNP setups with separate Conda environments, CMake build notes and smoke tests for scientific ML simulation interfaces.

### Opal Crystal Quality Image Analysis

Python image-analysis workflow for estimating local crystallinity in opal electron micrographs using particle detection, QC filtering, local hexatic order and robustness checks.
